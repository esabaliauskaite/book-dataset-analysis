# Books Dataset Analysis

## Goal

This project explores trends in book ratings, popularity, genres, and authors using exploratory data analysis (EDA), visualizations, and sentiment analysis techniques.

## Data Source

This project uses the **Best Books Ever Dataset** published on Zenodo.

### Citation

Rakshit, S., Cebrian, M., & Nikolaev, A. (2020). *Best Books Ever Dataset*. Zenodo. https://doi.org/10.5281/zenodo.4265096

- Dataset URL: https://zenodo.org/records/4265096
- License: Creative Commons Attribution-NonCommercial 4.0 International (CC BY-NC 4.0)

## Dataset Overview

The dataset contains information about books, including:

- Titles
- Authors
- Ratings
- Genres
- Number of reviews and votes
- Publication information

The raw data requires cleaning due to:

- Missing values
- Duplicate entries
- Inconsistent formatting
- Mixed categorical fields

## Objectives

- Perform exploratory data analysis (EDA)
- Analyze author popularity and rating trends
- Explore genre distributions
- Create interactive visualizations
- Investigate correlations between popularity metrics
- Experiment with sentiment analysis and NLP techniques
